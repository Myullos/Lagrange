# Curvature-Constrained Lagrange Interpolation

## 概要（Abstract）

本研究では、<b>ラグランジュ補間（Lagrange Interpolation）</b>における数値的不安定性、特に<b>ルンゲ現象（Runge phenomenon）</b>の抑制を目的として、補間手法の再構築を行った。対象関数としては代表的な試験関数である

[
f(x) = \frac{1}{1 + x^2}
]

を用いた。

まず、補間点として<b>チェビシェフノード（Chebyshev nodes）</b>を採用し、<b>大域的な振動の抑制</b>を図った。さらに、補間誤差を区間ごとに評価し、<b>積分誤差（面積誤差）に基づく局所的再補間（re-interpolation）</b>を導入することで、<b>誤差の大きい区間に対して重点的な補正</b>を行った。

次に、補間精度の向上のために、ノードの<b>y値そのものを最適化する手法</b>を導入した。具体的には、補間関数と真の関数との<b>二乗誤差積分の最小化</b>を目的としてランダム探索による調整を行い、<b>従来の補間よりも高精度な近似</b>を実現した。

さらに、補間関数の滑らかさに着目し、<b>曲率（2階微分）に基づく補正手法</b>を提案した。補間関数 ( P(x) ) に対し、( P''(x) ) を用いた補正項を加え、その係数を誤差最小化により決定することで、<b>振動を抑えつつ精度を向上させる枠組み</b>を構築した。

加えて、局所的な自由度を導入するために、区間ごとに定義される<b>バブル関数（bubble function）</b>を用いた補間を提案した。この関数は<b>補間点（節点）では常に0となり、節点間の内部でのみ値を持つ</b>という性質を持つため、<b>補間条件を保持したまま局所的な形状制御</b>を可能にする。本研究では、このバブル関数を用いて補間関数に局所補正項を加え、<b>節点を固定したまま区間内部のみを調整する手法</b>を構築した。

さらに、この補正に対して

[
\int (f''(x))^2 , dx
]

を最小化する最適化問題を設定することで、<b>不要な振動を抑制しつつ滑らかな補間</b>を実現した。

最後に、節点における傾きを変数とする<b>反復的傾き最適化手法（iterated slope optimization）</b>を構築した。この手法では、補間関数の導関数と節点傾きを自己一致させる反復計算を行うことで、<b>曲率を制御しつつ高精度な補間</b>を実現した。

数値実験の結果、これらの手法はいずれも従来のラグランジュ補間と比較して、<b>誤差の低減および振動の抑制に有効</b>であることが確認された。特に、<b>曲率に基づく最適化および局所補正を組み合わせた手法</b>は、<b>高い安定性と精度を両立する補間法</b>であることが示された。

本研究は、ラグランジュ補間を<b>「単なる補間手法」から「最適化問題として再定式化する枠組み」へ拡張</b>するものである。今後は理論的解析の深化および高次元問題への応用を進め、<b>研究としてさらに発展させていくことを目指す。</b>

---

## 背景

高次多項式補間は理論的には任意の連続関数を近似可能であるが、実際には端点付近で大きな振動（ルンゲ現象）が発生しやすい。この問題は数値解析・科学計算において古典的かつ重要な課題である。

従来はノードの選択（例：チェビシェフノード）による改善が主であったが、本研究ではさらに一歩進め、<b>補間そのものを最適化問題として捉える</b>というアプローチを採用した。

---

## 提案手法（Overview）

本研究では以下のアプローチを段階的に導入した：

* チェビシェフノードによる振動抑制
* 積分誤差に基づく局所再補間
* ノード値（y値）の最適化
* 曲率（2階微分）に基づく補正
* バブル関数による局所自由度の導入
* 曲率最小化問題としての定式化
* 傾き最適化に基づく反復手法

---

## 本研究の特徴

* <b>補間条件を保ったまま最適化が可能</b>
* <b>局所補正と大域的安定性の両立</b>
* <b>曲率最小化による滑らかさ制御</b>
* <b>補間問題の最適化問題への拡張</b>
