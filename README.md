# Curvature-Constrained Lagrange Interpolation

## 概要（Abstract）

本研究では、<b>ラグランジュ補間（Lagrange Interpolation）</b>における数値的不安定性、特に<b>ルンゲ現象（Runge phenomenon）</b>の抑制を目的として、補間手法の再構築を行った。対象関数としては代表的な試験関数である

$$
f(x) = \frac{1}{1 + x^2}
$$

を用いた。

まず、補間点として<b>チェビシェフノード（Chebyshev nodes）</b>を採用し、<b>大域的な振動の抑制</b>を図った。さらに、補間誤差を区間ごとに評価し、<b>積分誤差（面積誤差）に基づく局所的再補間（re-interpolation）</b>を導入することで、<b>誤差の大きい区間に対して重点的な補正</b>を行った。

次に、補間精度の向上のために、ノードの<b>y値そのものを最適化する手法</b>を導入した。具体的には、補間関数と真の関数との<b>二乗誤差積分の最小化</b>を目的としてランダム探索による調整を行い、<b>従来の補間よりも高精度な近似</b>を実現した。

さらに、補間関数の滑らかさに着目し、<b>曲率（2階微分）に基づく補正手法</b>を提案した。補間関数 ( P(x) ) に対し、( P''(x) ) を用いた補正項を加え、その係数を誤差最小化により決定することで、<b>振動を抑えつつ精度を向上させる枠組み</b>を構築した。

加えて、局所的な自由度を導入するために、区間ごとに定義される<b>バブル関数（bubble function）</b>を用いた補間を提案した。この関数は<b>補間点（節点）では常に0となり、節点間の内部でのみ値を持つ</b>という性質を持つため、<b>補間条件を保持したまま局所的な形状制御</b>を可能にする。本研究では、このバブル関数を用いて補間関数に局所補正項を加え、<b>節点を固定したまま区間内部のみを調整する手法</b>を構築した。

さらに、この補正に対して

$$
\int (f''(x))^2 , dx
$$

を最小化する最適化問題を設定することで、<b>不要な振動を抑制しつつ滑らかな補間</b>を実現した。

最後に、節点における傾きを変数とする<b>反復的傾き最適化手法（iterated slope optimization）</b>を構築した。この手法では、補間関数の導関数と節点傾きを自己一致させる反復計算を行うことで、<b>曲率を制御しつつ高精度な補間</b>を実現した。

数値実験の結果、これらの手法はいずれも従来のラグランジュ補間と比較して、<b>誤差の低減および振動の抑制に有効</b>であることが確認された。特に、<b>曲率に基づく最適化および局所補正を組み合わせた手法</b>は、<b>高い安定性と精度を両立する補間法</b>であることが示された。

本研究は、ラグランジュ補間を<b>「単なる補間手法」から「最適化問題として再定式化する枠組み」へ拡張</b>するものである。今後は理論的解析の深化および高次元問題への応用を進め、<b>研究としてさらに発展させていくことを目指す。</b>

---

## 背景

高次多項式補間は理論的には任意の連続関数を近似可能であるが、実際には端点付近で大きな振動（ルンゲ現象）が発生しやすい。この問題は数値解析・科学計算において古典的かつ重要な課題である。

従来はノードの選択（例：チェビシェフノード）による改善が主であったが、本研究ではさらに一歩進め、<b>補間そのものを最適化問題として捉える</b>というアプローチを採用した。

---

## 提案手法（Overview）

本研究では以下のアプローチを段階的に導入した：

* チェビシェフノードによる振動抑制
* 積分誤差に基づく局所再補間
* ノード値（y値）の最適化
* 曲率（2階微分）に基づく補正
* バブル関数による局所自由度の導入
* 曲率最小化問題としての定式化
* 傾き最適化に基づく反復手法

---

## 本研究の特徴

* <b>補間条件を保ったまま最適化が可能</b>
* <b>局所補正と大域的安定性の両立</b>
* <b>曲率最小化による滑らかさ制御</b>
* <b>補間問題の最適化問題への拡張</b>

---

### LIM_3_reLIM_exp1

- LIM_3_reLIM_exp1.py  
  大域的なラグランジュ補間に対して、誤差の大きい区間で局所的に再補間を行う**再補間手法（Re-interpolation）**の実装。

  主な手法は以下の通り：

  1. チェビシェフノードを用いて全体のラグランジュ補間 ( P(x) ) を構築  
  2. 各区間ごとに、元の関数との誤差（面積）を評価  
  3. 誤差が閾値を超える区間を検出  
  4. その区間内で新たに点を生成し、局所的なラグランジュ補間を構築  
  5. 必要な区間のみ補間を置き換え、精度を改善  

  特徴：
  - 誤差に応じた局所的な補間の再構築  
  - 不要な計算を避けた効率的な精度改善  
  - 大域補間と局所補間のハイブリッド構造  
  - 区間ごとの適応的精度制御が可能  

- LIM_3_animation.py  
  誤差に基づく区間選択と局所再補間の過程を可視化するManimアニメーション。

  主に以下を表示：
  - 初期ラグランジュ補間  
  - 誤差が大きい区間の特定  
  - 局所補間の追加による関数の改善

### LIM_4_y_optimization_exp2

- LIM_4_y_optimization_exp2.py  
  ノードの関数値（y値）を最適化することで補間誤差を低減する**y値最適化補間（y-value Optimization Interpolation）**の実装。

  主な手法は以下の通り：

  1. チェビシェフノードを用いて初期補間を構築  
  2. ノード位置は固定し、関数値（y値）のみを変化させる  
  3. 次の誤差を評価：
     
     $$
     \int_{-5}^{5} (f(x) - P(x))^2 , dx
     $$

  4. ランダム摂動を繰り返し適用  
  5. 誤差が減少する場合のみ更新を採用  

  特徴：
  - 補間点の位置を変えずに精度改善が可能  
  - 離散的なパラメータ最適化による補間制御  
  - 誤差の収束過程を評価可能  

- LIM_4_animation.py  
  y値最適化による補間関数の変化と誤差の減少過程を可視化するManimアニメーション。

  主に以下を表示：
  - 初期ラグランジュ補間からの変化  
  - ノード値（y値）の更新による関数形の変形  
  - 最適化の進行に伴う補間精度の改善

### LIM_5_curv_corr_exp3

- LIM_5_curv_corr_exp3.py  
  チェビシェフ補間に対して、曲率（2階微分）を用いた補正項を加える**曲率補正補間（Curvature-Corrected Interpolation）**の実装。

  主な手法は以下の通り：

  1. チェビシェフノードを用いてラグランジュ補間 ( P(x) ) を構築  
  2. 補間関数の2階微分 ( P''(x) ) を計算  
  3. 重み関数 ( w(x) ) を用いて補正項を定義  
  4. 次の形の補間関数を導入：
     
$$
P_{\text{corr}}(x) = P(x) + t , w(x) , P''(x)
$$

  5. 係数 ( t ) は、元の関数との二乗誤差積分が最小となるように決定

  特徴：
  - 曲率情報を直接利用したグローバル補正  
  - 1つのスカラー係数 ( t ) によるシンプルな制御  
  - ラグランジュ補間の振動（Runge現象）を抑制

### LIM_6_curv_driven_interp_exp4

- LIM_6_curv_driven_interp_exp4.py  
  局所的なバブル関数（bubble function）を用いた**曲率駆動補間（Curvature-Driven Interpolation）**の実装。

  主な手法は以下の通り：

  1. **区分線形補間（piecewise linear）**を初期近似として構築  
  2. 各区間ごとに局所的な補正関数（bubble関数）を導入  
  3. 各バブル係数を最適化し、  
     - 曲率（2階微分）の二乗積分  
     - ラグランジュ補間との乖離  
     を同時に最小化  
  4. これにより、滑らかさと安定性を両立した補間関数を構成  

  特徴：
  - 局所補正によりノード値を保ったまま形状を調整可能  
  - 振動（Runge現象）の抑制  
  - グローバル補間とローカル補正のハイブリッド構造  

- LIM_6_animation.py  
  曲率駆動補間の構築過程および補間結果を可視化するManimアニメーション。

  主に以下を表示：
  - 区分線形補間からの変形過程  
  - ラグランジュ補間との比較  
  - 曲率補正による滑らかさの改善

### LIM_7_curv_controlled_interp_exp5

- LIM_7_curv_controlled_interp_exp5.py  
  傾き（一次導関数）を反復的に更新することで曲率を制御する**曲率制御補間（Curvature-Controlled Interpolation）**の実装。

  主な手法は以下の通り：

  1. ノード間の差分から初期傾き（スロープ）を推定  
  2. ノード値と傾きに基づき、区分三次補間（cubic spline）を構築  
  3. 補間関数の導関数を再評価し、傾きを更新  
  4. 傾きが自己無撞着（self-consistent）になるまで反復  

  この反復過程により、補間関数の曲率が自然に調整される。

  特徴：
  - 曲率を直接最適化せず、傾きの自己整合性から間接的に制御  
  - ノード値を厳密に保持（補間条件を維持）  
  - 振動の抑制と滑らかさの両立  
  - グローバル多項式（ラグランジュ）に依存しない安定な構造  

- LIM_7_animation.py  
  傾きの反復更新による補間関数の収束過程を可視化するManimアニメーション。

  主に以下を表示：
  - 初期スロープからの変化  
  - 反復による収束の様子  
  - ラグランジュ補間との誤差比較
